# node-webrender starter

## Install
```
npm install
```

## Start with hot-reload
```
npm run dev
```

## Build
```
npm run build
```

## Start (after build)
```
npm start
```

## Package (after build)
```
npm run package
```

## react-devtools
Work even with packaged app unless you've done `npm i --production`
```
npx react-devtools
```
